# Loading button
Button that provides the ability for theming and its loading state.

### Usage

```html
<matxtnd-loading-button mat-raised-button [busy]="busy" color="primary" [disabled]="disabled" (submit)="onSubmit()">
  <matxtnd-loading-button-ready>
    Update
  </matxtnd-loading-button-ready>
  <matxtnd-loading-button-busy>
    Updating...
  </matxtnd-loading-button-busy>
</matxtnd-loading-button>
```
